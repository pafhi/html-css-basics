Listas

Para começar, você pode criar listas não-ordenadas:

  • Primeiro insira a tag "ul", indicando que a lista não é ordenada;
  • Em seguida, crie a tag "li" para indicar cada elemento da lista;
  • Pronto! Você criou uma lista desordenada.

Já as listas ordenadas são feitas da seguinte forma:

  1. Insira a tag "ol", que significa ordened list;
  2. Crie tags "li" para indicar cada elemento da lista;
  3. E voila, você criou uma lista ordenada!;

Importante: Apesar de parecidas para nós humanos, as listas ordenadas e desordenadas são completamente distintas para o navegador.
