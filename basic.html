Listas

Para começar, você pode criar listas não-ordenadas:

  • Primeiro insira a tag <ul>, indicando que a lista não é ordenada;
  • Em seguida, crie a tag <li> para indicar cada elemento da lista;
  • Pronto! Você criou uma lista desordenada.

Já as listas ordenadas são feitas da seguinte forma:

  1. Insira a tag <ol>, que significa ordened list;
  2. Crie tags <li> para indicar cada elemento da lista;
  3. E voila, você criou uma lista ordenada!;

Não menos importantes, as listas explicativas podem ser criadas dessa forma:

Adicione a tag <dl>.
Para criar a lista de definições;
Dentro da tag <dl> adicione <dt>
Para inserir a palavra que precede a explicação;
Por fim, adicione <dd>
Para adicionar a explicação.

Importante: Apesar de parecidas para nós humanos, as listas ordenadas e desordenadas são completamente distintas para o navegador.

Elementos inline

Chamamos de elementos inline, às tags que atuam dentro de um parágrafo, como "strong" e "em".

Outro elemento inline muito importante é a tag <a> que indica links quando sucedida por "href = "..."". Por exemplo, caso queiramos adicionar um link para a página do google , devemos escrever:

<a href = "http://google.com">www.google.com</a>

E o resultado será: www.google.com.
